# MergeMusic
结合了网易云音乐和qq音乐基于单页面的音乐播放器。

功能
* [x] 网易云音乐和QQ音乐混合播放
* [x] 音乐歌词图片播放及下载
* [x] 歌词音频可视化
* [x] 播放列表顺序选择
* [ ] 播放列表管理
* [ ] 歌单专辑特殊处理

### 基于以下模块
* Vue
* BootStrap
* Hitokoto网易云api
* QQ音乐原生api
* 阿里云函数计算
